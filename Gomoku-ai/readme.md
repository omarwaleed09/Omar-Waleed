## How to Run

Clone the repository:

```bash
git clone https://github.com/omarwaleed09/gomoku-ai.git
cd gomoku-ai
```

Run the game:

```bash
python GOMOKU.py
```

Choose mode:

- Press `1` for Human vs AI  
- Press `2` for AI vs AI
